# Page Design: Battle Lobby / Matchmaking

## 1. UX Goal
**"Anticipation"**
Build hype while finding an opponent. Eliminate boredom during wait times.

## 2. Sections
1.  **Player Card**: Your avatar, ready state.
2.  **Opponent Card**: Searching state -> Found state.
3.  **VS Badge**: Central visual anchor.
4.  **Tip Carousel**: Useful info while waiting.

## 3. Components
-   **VS Badge**:
    -   Large "VS" text with `Electric Violet` and `Spark Orange` clash.
-   **Searching Animation**:
    -   Radar sweep or Avatar cycling effect.
    -   Text: "Scanning for worthy opponents..."
-   **Found State**:
    -   Sound: Siren/Alert.
    -   Show Opponent Avatar + Name + Level.
    -   Countdown: "Battle starts in 3..."

## 4. Primary User Actions
1.  **Cancel Search**: Small, bottom right (don't make it too easy to quit).
2.  **Ready Up**: (Optional, if manual trigger needed).

## 5. Feedback & Interaction
-   **Pulse**: The VS badge pulses with the beat of the background music (if any) or imaginary heartbeat (100bpm).
-   **Match Found**: Screen flash `Electric Violet`.

## 6. Color References
-   **Background**: Darker shade of `Cloud Mist` or `Electric Violet` gradient overlay to distinct from "Learning" mode.
-   **VS Text**: `Spark Orange` (#F59E0B).
