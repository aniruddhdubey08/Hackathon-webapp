# Page Design: Home / Landing Page

## 1. UX Goal
**"Get Started in 3 Seconds"**
The goal is to explain the value proposition instantly and drive the user to "Start Learning" or "Login". Avoid aggressive marketing. Focus on "Fun" and "Free".

## 2. Sections
1.  **Hero Section**: Value prop + CTA.
2.  **Feature Highlights**: 3 key benefits (Gamified, Competitive, Free).
3.  **Social Proof**: "Join X students".
4.  **Footer**: Minimal links.

## 3. Components
-   **Hero Card**:
    -   *Background*: `Cloud Mist` with a decorative blob in `Electric Violet` (low opacity).
    -   *Headline*: "Master Your Subjects, One Battle at a Time." (H1, `Ink Slate`).
    -   *Subtext*: "Gamified learning for students who want to win." (`Stone Gray`).
    -   *CTA Button*: `Electric Violet` (Solid), Text "Start Battling Now".
    -   *Image*: Illustration of a student holding a trophy or shield.
-   **Feature Cards**:
    -   *Layout*: 3-column grid (1-column on mobile).
    -   *Icon*: Large, `Spark Orange`.
    -   *Title*: H3, `Ink Slate`.
    -   *Text*: `Stone Gray` (short).

## 4. Primary User Actions
1.  **"Start Battling Now"**: Directs to Signup/Login.
2.  **"Login"**: Secondary top-right link for returning users.

## 5. Feedback & Interaction
-   **Hero CTA**: On hover, scale 1.05x and brighten shadow. "Pulse" animation every 5s if idle.
-   **Scroll**: Reveal Feature Cards with a slide-up animation.

## 6. Color References
-   **Background**: `Cloud Mist` (#F8FAFC)
-   **Primary Button**: `Electric Violet` (#6366F1)
-   **Headings**: `Ink Slate` (#1E293B)
