# Page Design: Home / Landing (Cyber Command)

## 1. UX Goal
**"Recruitment into the Future"**
The user is not just "signing up", they are "initializing". High-tech, mysterious, and elite.

## 2. Sections
1.  **Hero Frame**: "DOMINATE THE INTELLECTUAL FRONTIER".
2.  **Mission Objectives**: 3 key benefits (Upgrade, Compete, Conquer).
3.  **Global Telemetry**: Showing "Active Agents" (Users online).
4.  **Footer**: "System Status: Online".

## 3. Components
-   **Hero Module**:
    -   *Background*: Void Black with low-opacity grid.
    -   *Headline*: "OPERATOR: READY?" (Glitch effect text).
    -   *Subtext*: "Deploy advanced cognitive protocols." (`Terminal Gray`).
    -   *Call to Action*: "INITIALIZE UPLINK" (Primary Neon Cyan Button).
-   **Objective Cards**:
    -   Glass-panel cards.
    -   Icons: Neon wireframe style.
    -   *Hover*: Border glows Cyan.

## 4. Primary User Actions
1.  **"INITIALIZE UPLINK"**: Main entry (Login/Signup).
2.  **"SYSTEM STATUS"**: Check server status (just a cool link to features).

## 5. Feedback & Interaction
-   **Cursor**: Custom crosshair cursor (optional but recommended).
-   **Hover Effect**: Elements slide up + Text color shifts to Cyan.
-   **Load**: "Terminal typing" effect for headlines.

## 6. Color References
-   **Primary**: `Neon Cyan` (#06b6d4)
-   **Background**: `Void Black` (#020617)
