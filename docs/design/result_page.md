# Page Design: Mission Debrief (Result)

## 1. UX Goal
**"After Action Report"**
Clinical analysis of performance.

## 2. Sections
1.  **Outcome Stamp**: "MISSION SOLVED" or "MISSION FAILED".
2.  **Rewards**: Credits/XP scrolling numbers.
3.  **Telemetry**: Graph of accuracy over time.

## 3. Components
-   **Outcome Stamp**: Stencil font, angled boarder.
-   **XP Bar**: Fills up with liquid Cyan animation.
-   **Action Panel**: "RE-DEPLOY" (Play Again) or "RETURN TO BASE".

## 4. Primary User Actions
1.  **Re-Deploy**: Primary Action.
2.  **Log Out**: Secondary.

## 5. Feedback & Interaction
-   **Victory**: "MISSION SUCCESS" voiceover (optional).
-   **Defeat**: "SYSTEM REBOOT REQUIRED".

## 6. Color References
-   **Success**: `Neon Cyan`
-   **Fail**: `Terminal Gray`
