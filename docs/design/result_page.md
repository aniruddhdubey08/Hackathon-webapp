# Page Design: Battle/Quiz Result

## 1. UX Goal
**"Celebration or Encouragement"**
Make winning feel epic. Make losing feel like a "Challenge to Improve".

## 2. Sections
1.  **Outcome Banner**: "Victory!", "Defeat", or "Level Complete".
2.  **Reward Summary**: XP Gained, Coins earned.
3.  **Detailed Stats**: Accuracy, Time, Streak maintained.
4.  **CTA**: "Play Again" or "Return to Map".

## 3. Components
-   **Victory Banner**:
    -   Animated Confetti.
    -   Text: Gold gradient `Spark Orange`.
    -   Character: Jumping/Cheering.
-   **Stats Grid**:
    -   Clean layout showing numbers + labels.
-   **XP Bar Animation**: 
    -   Current Level bar filling up with new XP. Sound: "Ding ding ding".

## 4. Primary User Actions
1.  **Continue**: Main `Electric Violet` button.
2.  **Rematch**: (For battles) Secondary button.
3.  **Review Answers**: Text link.

## 5. Feedback & Interaction
-   **Entrance**: Elements cascade in (staggered animation).
-   **Confetti**: Canvas overlay particle effect.

## 6. Color References
-   **Victory**: `Spark Orange` (#F59E0B) + `Success Fern`.
-   **Defeat**: `Stone Gray` (Don't use Red for the whole screen, it feels punishing). Use `Electric Violet` for "Try Again".
